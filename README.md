# house-price-prediction

1. Project Description

This project focuses on predicting the selling price of houses in US dollars using a machine learning model based on CatBoost.

The goal is to analyze housing data and build a model that can accurately estimate property prices based on various features describing the property.

2. Features

The dataset includes the following features:

LotArea — lot size in square feet
LotArea_M — lot size in square meters
Street — type of road access
BldgType — type of dwelling
OverallQual — overall material and finish quality
OverallCond — overall condition rating
YearBuilt — original construction date
YearRemodAdd — remodeling date
RoofStyle — type of roof
ExterQual — exterior material quality
ExterCond — present condition of exterior materials
Foundation — type of foundation
TotalBsmtSF — total basement area (sq ft)
TotalBsmtSF_M — total basement area (sq meters)
Heating — type of heating
HeatingQC — heating quality and condition
CentralAir — central air conditioning
GrLivArea — above-ground living area (sq ft)
GrLivArea_M — above-ground living area (sq meters)
Bath — number of bathrooms
KitchenQual — kitchen quality
GarageArea — garage area (sq ft)
GarageArea_M — garage area (sq meters)
DateSold — month and year of sale
SaleCondition — sale conditions

3. Target Variable
SalePrice — house selling price in USD
